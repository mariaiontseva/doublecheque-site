# DoubleCheque Email Signature

This folder hosts the Gmail-ready DoubleCheque signature assets on the public website.

## Public URLs

After the site is deployed, the signature file is available at:

`https://doublecheque.com/email-signature/signature.html`

The image assets are available at:

- `https://doublecheque.com/email-signature/assets/farid-signature-photo.jpg`
- `https://doublecheque.com/email-signature/assets/dc-mark-dark.png`
- `https://doublecheque.com/email-signature/assets/sig-linkedin.png`
- `https://doublecheque.com/email-signature/assets/sig-instagram.png`

## Gmail Setup

1. Open `https://doublecheque.com/email-signature/signature.html` in Chrome.
2. Select the rendered signature, not the raw HTML.
3. Copy it.
4. In Gmail, open Settings > See all settings > General > Signature.
5. Paste it into the signature box.
6. Set the signature defaults for new emails and replies.
7. Save changes.

## Notes

- `signature.html` is the Gmail source of truth.
- `signature-preview.dc.html` is only the design reference.
- Keep the table layout and inline styles intact; email clients rely on them.
