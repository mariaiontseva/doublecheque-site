# DoubleCheque — investment website

Static prototype exported from Claude Design (claude.ai/design) and served as-is.

- `index.html` — the design (DoubleCheque.dc.html)
- `support.js` — Claude Design runtime that renders the `<x-dc>` template
- `image-slot.js` — image placeholder custom element
- `farid-gasanov.jpg` — team photo

Live via GitHub Pages. This is a design prototype, not production code.
